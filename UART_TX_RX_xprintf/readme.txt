Базовый проект-пример.

Эхо UART посимвольно, плюс вывод отладочной информации с помощью xprintf

Адаптирован для работы на плате с программатором.
В проекте прерывание настраивается на работу от кнопки. 
По её нажатии происходит переключение состояния зелёного светодиода.
В основном потоке приложение мигает красным светодиодом.
