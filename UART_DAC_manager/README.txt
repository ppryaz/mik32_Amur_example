Управление ЦАП через команды, получаемые через UART
Базовый проект-пример MIK32 АМУР  K1948ВК018
Адаптирован для работы на плате с программатором v0.3.

Проект демонстрирует возможность управления ЦАПом микроконтроллера с помощью команд, отправляемых по  интерфейсу UART.

Формат строки для отправки команд: 12001. Первые четыре разряда это устанавливаемое напряжение в милливольтах, последняя цифра - это номер канала.
В этой версии программы отсутствует проверка корректности отправляемых из терминала строк.
Допустимые значения - первые четыре цифры: от 0 до 1200, последняя цифра 1 или 2 канала
